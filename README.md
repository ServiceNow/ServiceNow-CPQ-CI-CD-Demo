# ServiceNow-CPQ-CI-CD-Demo
## Purpose
The is an example of how Git and CI/CD can be used with ServiceNow CPQ to establish strong version control using best in class tooling from GitHub. 

## What does it do
This code utilizes github actions to pull objects from and push objects to your ServiceNow CPQ instances for the purposes of version control, promotion, and conflict management. 
For a full overview of current implementation see the [wiki](https://github.com/ServiceNow/ServiceNow-CPQ-CI-CD-Demo/wiki).
